Welcome to Brightbox

A lightweight but solid jQuery lightbox plugin.

Features

Lightweight
Only 3.2 KB (minified), no images except for an optional loader.gif
Solid
Sane default behavior, works in every browser from IE7+, Keyboard navigation
Flexible
Customizeable: All styling can be tweaked by providing CSS styles.
Easy setup
Just call brightbox() on a selection of thumbnail links

How to use

HTML
<!-- Include stylesheet, jquery and plugin -->
<link rel="stylesheet" type="text/javascript" href="brightbox.css" />
<script src="jquery.min.js"></script>
<script src="brightbox.js"></script>

<!-- Markup for images-->
<ul id="images">
<li><a href="path/to/image.jpg"><img src="path/to/thumbnail.jpg" /></a></li>
<li><a href="path/to/another/image.jpg" title="Optionally give a description here"><img src="path/to/another/thumbnail.jpg" /></a></li>
</ul>
Javascript
$(document).ready(function){
	$('#images a').brightbox({
		/* options */
	});
});
