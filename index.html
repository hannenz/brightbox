Welcome to Brightbox

A lightweight but solid jQuery lightbox plugin.

Features

Lightweight
Only xxx Kb, no images except for an optional loader.gif
Solid
Sane default behavior, works in every browser from IE7+
Easy setup
Just call brightbox() on a selection of thumbnail links

How to use

HTML
<link rel="stylesheet" type="text/javascript" href="brightbox.css" />
<script src="brightbox.js"></script>

<ul id="images"></ul>
<li><a href="path/to/image.jpg"><img src="path/to/thumbnail.jpg" /></a>
Javascript
$(document).ready(function){
	$('#images a').brightbox({
		/* options */
	});
});

Options

boolean Whethet or not a click on the overlay closes the box (default: true)
